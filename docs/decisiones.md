# Decisiones y Desafíos

## Decisiones
- Se utilizó sockets TCP por ser una forma confiable de comunicación.
- Se utilizó JSON para estructurar los mensajes entre cliente y servidor.

## Desafíos
- Manejo de múltiples clientes: se resolvió usando hilos.
- Obtención de CPU y RAM: se resolvió usando la librería psutil.
